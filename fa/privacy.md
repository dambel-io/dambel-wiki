# سیاست حریم خصوصی

در دمبل، حریم خصوصی شما برای ما اهمیت دارد. در ادامه توضیح می‌دهیم که با اطلاعات شما چه می‌کنیم.

**اطلاعاتی که جمع‌آوری می‌کنیم**

هنگام استفاده از دمبل، اطلاعاتی که خودتان وارد می‌کنید - مانند نام، ایمیل، شماره تلفن و جزئیات پروفایل - به همراه داده‌های ورزشی و تغذیه‌ای که ثبت می‌کنید را نگه می‌داریم. همچنین آمار کلی استفاده از اپ را برای بهبود سرویس جمع‌آوری می‌کنیم.

**نحوه استفاده**

از اطلاعات شما برای اجرای اپ، شخصی‌سازی تجربه‌تان، ارسال اعلان‌های مرتبط و بهبود سرویس استفاده می‌کنیم. از داده‌های سلامت یا ورزشی شما برای تبلیغات استفاده نمی‌کنیم.

**اشتراک‌گذاری**

داده‌های شما را نمی‌فروشیم. فقط با ارائه‌دهندگان خدماتی که برای اجرای اپ ضروری هستند (مانند سرویس‌های ابری و API های هوش مصنوعی) به اشتراک می‌گذاریم، یا در صورت الزام قانونی.

**امنیت**

از اتصالات رمزگذاری‌شده و استانداردهای امنیتی صنعت برای حفاظت از اطلاعات شما استفاده می‌کنیم.

**حقوق شما**

هر زمان می‌توانید درخواست دسترسی، اصلاح یا حذف اطلاعات خود را از طریق support@dambel.io ارسال کنید. ظرف ۳۰ روز پاسخ خواهیم داد.

**به‌روزرسانی**

ممکن است این سیاست را گاهی به‌روز کنیم. در صورت تغییر، از طریق اپ اطلاع‌رسانی می‌کنیم.

سوالی دارید؟ با **support@dambel.io** در تماس باشید.
